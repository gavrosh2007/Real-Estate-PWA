1. Для посетителей сайта

Какие данные собираются:
Сайт не собирает личные данные автоматически. Техническая информация (IP-адрес, тип браузера, устройство) обрабатывается хостинг-провайдером для обеспечения работы сайта и защиты от атак. Эта информация не передаётся третьим лицам.

Файлы cookie:
Используются только технические cookie (сохранение языка, настроек). Рекламные и трекинговые cookie не используются.

Формы связи (если есть):
Данные, отправленные через формы (имя, email, сообщение), используются только для ответа на запрос и не передаются третьим лицам.

2. Для кандидатов / апликантов

Какие данные собираются при приёме на работу:
Имя, контакты, резюме, информация об образовании и опыте работы.

Цель сбора:
Рассмотрение кандидатуры, проведение собеседований, принятие решения о найме.

Хранение:
Данные кандидатов хранятся не дольше 6 месяцев после завершения процесса найма, если кандидат не дал письменное согласие на включение в кадровый резерв.

Права кандидатов:
